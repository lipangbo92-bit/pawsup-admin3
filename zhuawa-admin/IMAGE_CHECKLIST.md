## 常见问题排查

### 图片不显示的可能原因

1. **图片路径错误**
   - 检查是否为完整的云存储 URL
   - 检查是否包含 `https://`

2. **图片权限问题**
   - 云存储图片是否设置了公开访问权限

3. **图片格式问题**
   - 检查图片格式是否为 JPG、PNG 等常见格式
   - 检查图片是否损坏

4. **网络问题**
   - 检查网络连接
   - 检查图片 URL 是否可访问

5. **字段为空**
   - 检查数据库中图片字段是否有值
   - 检查是否有默认值处理

## 图片显示最佳实践

1. **使用 wx:if 判断图片是否存在**
   ```xml
   <image wx:if="{{item.image}}" src="{{item.image}}" mode="aspectFill"/>
   <view wx:else>默认图标</view>
   ```

2. **设置默认图片**
   - 准备默认头像、默认服务图等

3. **图片裁剪模式**
   - 头像使用 `mode="aspectFill"`
   - Banner 使用 `mode="aspectFill"`
   - 详情图使用 `mode="widthFix"`

4. **图片加载失败处理**
   - 使用 `binderror` 事件处理加载失败
   - 显示默认图片
